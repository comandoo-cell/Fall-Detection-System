# Benchmarks Package

This directory contains performance benchmarks and accuracy tests for the fall detection system.

## Running Benchmarks

```bash
python benchmarks/run_benchmarks.py
```

## Benchmark Results

Results are saved to `benchmark_results.json` with detailed metrics including:
- Overall accuracy
- Precision, Recall, F1-Score
- Confusion matrix
- Processing speed (FPS)
- Edge case test results
